Easy Breathe

Firmware Division

Electronic masks currently being used in the medical field can be uncomfortable and noisy. Easy Breathe provides an electronic-based face mask designed for medical workers that solves these problems. It utilizes an optimized human machine interface with LEDâ€™s, switch buttons, and a buzzer; the user is given complete control over the fan speed with the buttons, and is notified when the battery is low. The software team developed and implemented the logic that makes the mask run, controlling the fan speed and making it easy for the user to check battery level.

Future work includes software updates to prolong the battery life, improve the user interface, and support any future components added by the hardware team.
